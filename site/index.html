八个一起干活的子命令

共享同一套配置与日志机制。

clean

清洗数据

在工作簿内部清理：去空行空列、去首尾空格、规范表头、按列去重、排序、按列拆分。输出到独立目录，原文件从不改动。

files

整理文件

按文件名正则 / 修改日期 / 内容标记列分类到子文件夹；用模板批量改名（冲突自动加序号）；按内容哈希去重——重复文件移动到 duplicates/，从不删除。

merge

合并工作簿

把目录下所有导出合并成一个新工作簿：append 按表头并集对齐，sheets 每文件一个工作表，join 按关键列跨表连接。输出从不覆盖已有文件。

validate

校验数据

按声明式规则校验与对账：必填列 / 唯一键 / 非负 / 总计核对，加正则、数值范围、允许值、日期、列间比较；支持跨文件一致性与外键核对，问题可导出成待办工作簿逐条处理。

report

汇总报表

把多个导出聚合成汇总报表：多级分组、交叉透视、小计与总计，十种聚合函数（求和 / 计数 / 均值等）；跨文件出一份总表，或逐文件各出一份。

compare

对比两期

按关键列对比两个工作簿或两期数据，输出新增 / 删除 / 变更三张结果表——变更逐单元格列出旧值与新值；数值可设容差。

split

拆分工作簿

把一个工作簿拆成多个文件：按列值分组、按固定行数分块、或每个工作表一个文件。输出从不覆盖已有文件。

profile

数据画像

只读摸底新导出：每列一行，给出推断类型、缺失数与占比、去重值数、最小 / 最大 / 均值和示例值——写清洗或校验规则之前，先看清数据长什么样。

默认安全，逐步可控

破坏性操作前先预览，再确认。

默认 dry-run

文件操作默认只预览计划，加 --apply 才真正移动。

去重只移动

重复文件移动到 duplicates/，永不删除；命名冲突自动改唯一名。

兼容性扫描

含公式 / 图表 / 合并单元格 / 宏的文件默认跳过，并说明原因。

可审计报告

每次运行产出 JSON / 文本报告，含处理与跳过的文件。

常见问题

支持 macOS / Linux 吗？

是。它是纯 Python、跨平台的，不依赖本地 Excel 或 Windows COM。Windows 提供免安装 exe；macOS / Linux 的源码构建请联系我们获取。

我的数据会被上传吗？

不会。处理全程在本机完成。唯一的联网路径是你手动触发的检查更新。

免费版和 Pro 有什么区别？

功能完全一致。免费版每次运行最多处理 20 个文件，Pro 解除该上限。许可是离线签名文件，无需账号或激活服务器。

原文件会被改动吗？

不会。输入始终只读；文件操作默认只预览，去重只移动不删除，破损 / 加密文件会被记录并跳过。
