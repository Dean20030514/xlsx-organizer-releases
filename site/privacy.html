隐私政策

生效日期：2026-06-13

1. 概述

xlsx-organizer 是一款离线工具。在设计上，你的电子表格数据不会离开你的设备。本政策说明软件如何处理数据，以及在你购买 Pro 时我们会收集什么信息。

2. 软件在本机处理的数据

clean / files / merge 三个功能全部在你的设备本地完成；软件不会将你的文件内容上传到任何服务器。运行历史、预设、许可文件等都保存在你本机的 ~/.xlsx-organizer/ 目录下，由你掌控，可随时删除。

若你购买并安装 Pro 许可证，软件会读取本机一个稳定的机器标识符（在 Windows 上为系统的 MachineGuid），用于将许可证绑定到你的机器（"节点锁"）。软件只会保存并使用该标识符经加盐后的单向哈希（指纹）；原始标识符不会被存储或上传。该指纹仅用于离线校验本机是否在许可范围内。

3. 唯一的联网行为：检查更新

仅当你主动运行 xlsx-organizer update check 或在图形界面点击"检查更新"时，软件才会向 GitHub 发起一次请求，询问最新发布版本。该请求：

  • 只发送你的软件版本号（作为 User-Agent，如 xlsx-organizer/0.6.0）；
  • 与任何网络请求一样，目标服务器（GitHub）会看到你的 IP 地址；
  • 不发送你的任何文件内容、文件名或个人信息。

软件没有任何自动或后台联网、没有遥测、没有使用统计、没有广告或第三方跟踪。

4. 购买 Pro 时收集的信息

为签发 Pro 许可文件，我们会收集你提供的姓名和邮箱（以及席位数）。这些信息会写入你的签名许可文件，并由许可方保留，仅用于：

  • 签发与（必要时）重发你的许可文件；
  • 开具发票或收据；
  • 提供与购买相关的支持。

为实现机器绑定（节点锁），你在购买时还会通过 xlsx-organizer license fingerprint 提供每个席位对应机器的指纹哈希；该哈希会写入你的签名许可文件，作为该许可证可在哪些机器上生效的依据。更换硬件后请联系支持以重新签发。

我们不出售这些信息，也不将其用于与上述无关的目的。

5. 保留与你的权利

购买相关信息保留至履行支持、开票与法律义务所需的期限。你可通过联系邮箱要求查询、更正或删除你的购买信息。

6. 本网站

本营销网站为静态页面，不设置跟踪 Cookie，也不嵌入第三方分析或字体。你的语言偏好仅以浏览器本地存储（localStorage）保存在你自己的设备上。

Privacy Policy

Effective date: 2026-06-13

1. Overview

xlsx-organizer is an offline tool. By design, your spreadsheet data never leaves your device. This policy explains how the software handles data and what we collect when you buy Pro.

2. Data the software processes locally

The clean / files / merge features all run locally on your device; the software does not upload your file contents to any server. Run history, presets, and the license file are stored under ~/.xlsx-organizer/ on your own machine, under your control, and can be deleted at any time.

If you buy and install a Pro license, the software reads one stable machine identifier (on Windows, the system's MachineGuid) to bind the license to your machine (a "node-lock"). It stores and uses only a salted, one-way hash (a fingerprint) of that identifier; the raw identifier is not stored or transmitted. The fingerprint is used only to check, offline, whether this machine is covered by the license.

3. The only network activity: update check

Only when you explicitly run xlsx-organizer update check or click "Check for updates" in the GUI does the software make a single request to GitHub asking for the latest released version. That request:

  • sends only your software version (as a User-Agent, e.g. xlsx-organizer/0.6.0);
  • like any network request, lets the destination (GitHub) see your IP address;
  • does not send any of your file contents, file names, or personal information.

There is no automatic or background networking, no telemetry, no usage analytics, and no ads or third-party tracking.

4. Information collected when you buy Pro

To issue a Pro license file, we collect the name and email you provide (and a seat count). This information is written into your signed license file and retained by the Licensor solely to:

  • issue and, if needed, re-issue your license file;
  • provide invoices or receipts;
  • provide purchase-related support.

To enable machine binding (node-lock), you also provide a fingerprint hash for each seat's machine via xlsx-organizer license fingerprint; that hash is written into your signed license file to record which machines the license is valid on. If your hardware changes, contact support for a re-issue.

We do not sell this information or use it for unrelated purposes.

5. Retention & your rights

Purchase information is retained for as long as needed to meet support, invoicing, and legal obligations. You may request access to, correction of, or deletion of your purchase information via the contact email.

6. This website

This marketing site is a static page. It sets no tracking cookies and embeds no third-party analytics or fonts. Your language preference is stored only in your browser's localStorage, on your own device.

7. Contact

For privacy matters, contact deanymrq@gmail.com.
