# xlsx-organizer — releases

Download the latest `xlsx-organizer.exe` from the [Releases page](https://github.com/Dean20030514/xlsx-organizer-releases/releases).

xlsx-organizer is an offline, bilingual (中文/English) Excel/CSV batch tool: clean data inside workbooks, organize workbook files, and merge many exports into one. Your files never leave your machine. The in-app `update check` command reads this repository's latest release tag — nothing is ever downloaded automatically.

The source repository is private; this repository hosts release artifacts only.
